## Требования

Убедитесь, что у вас установлен и запущен **Floral**. Скриптинг работает прямо из коробки и не требует установки сборщиков или внешних утилит.

## Структура директорий

Все пользовательские скрипты Lua располагаются в следующей папке:

```text
.minecraft/config/floral/scripts/
```

<Note>
  **Hot Reloading**: Каждое сохранение файла `.lua` мгновенно перезагружает скрипт прямо в запущенном клиенте.
</Note>

## Создание первого модуля

<Steps>
  <Step title="Перейдите в папку скриптов">
    Откройте директорию `.minecraft/config/floral/scripts/` на вашем компьютере.
  </Step>
  <Step title="Создайте Lua-файл">
    Создайте текстовый файл `my_hud_module.lua`.
  </Step>
  <Step title="Добавьте исходный код">
    Вставьте пример кода для регистрации интерактивного HUD-модуля:

    <CodeGroup>
      ```lua my_hud_module.lua
      -- Регистрируем наш модуль в категории HUD (отобразится в меню Floral)
      local module = client:register_module("Custom HUD", "Отображает FPS и ник игрока", "HUD")

      -- Создаем слайдер для регулировки шрифта прямо из игры
      local scale_setting = module:add_slider("HUD Scale", "Размер шрифта", 8, 20, 11, 1)

      -- Вспомогательные переменные для оптимизации (чтобы не перегружать ПК вычислениями каждый кадр)
      local cached_username = ""
      local cached_text = ""
      local cached_width = 120
      local last_update_time = 0.0

      -- Метод on_render вызывается каждый игровой кадр
      module:on_render(function()
          if not minecraft:is_in_world() then return end

          local current_time = os.clock()
          local scale = scale_setting:getValue()

          -- 1. Получаем никнейм игрока один раз при старте
          if cached_username == "" then
              cached_username = minecraft:get_username()
          end

          -- 2. Обновляем FPS и пересчитываем ширину рамки раз в 150мс (вместо каждого кадра)
          if current_time - last_update_time > 0.15 or cached_text == "" then
              cached_text = "FPS: " .. tostring(minecraft:get_fps()) .. "  |  User: " .. cached_username
              cached_width = renderer:get_text_width(cached_text, scale)
              last_update_time = current_time
          end

          -- 3. Высчитываем размеры рамки с учетом отступов
          local width = cached_width + 24
          local height = scale + 18

          -- 4. Включаем перетаскивание (100, 100 — это координаты по умолчанию)
          local session = renderer:begin_drag("floral.custom_fps_hud", 100, 100, width, height)
          local x = session:positionX()
          local y = session:positionY()

          -- Открываем настройки при нажатии правой кнопкой мыши на HUD
          if session:rightClicked() then
              module:open_settings(x, y + height + 6, 200)
          end

          -- Рисуем рамку вокруг HUD при перемещении
          if session:dragging() then
              renderer:render_drag_outline(session, 8, 1.0)
          end

          -- 5. Рисуем фон с красивым размытием
          local color_bg = renderer:rgba(18, 18, 22, 195)
          local color_primary = renderer:get_client_primary_color()

          renderer:prepare_blur(6.0)
          renderer:draw_blur(x, y, width, height, 8)
          renderer:draw_rect_rounding(x, y, width, height, 8, color_bg)

          -- 6. Отрисовываем текст ровно по центру рамки
          renderer:draw_text_centered_in_rect(cached_text, x, y, width, height, color_primary, scale)

          -- 7. Завершаем сессию перетаскивания
          renderer:end_drag(session)
      end)
      ```
    </CodeGroup>
  </Step>
  <Step title="Проверьте результат в игре">
    Сохраните файл `Ctrl + S`. Floral мгновенно активирует ваш новый модуль в игре!
  </Step>
</Steps>

<Tip>
  Перейдите к [Спецификации Lua API](/lua-api), чтобы узнать обо всех доступных функциях отрисовки текста, текстур, блюра и получения игровых данных.
</Tip>
