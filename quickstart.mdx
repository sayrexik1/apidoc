## Требования

Убедитесь, что у вас установлен и запущен **Floral**. Скриптинг работает прямо из коробки и не требует установки сборщиков или внешних утилит.

## Структура директорий

Все пользовательские скрипты Lua располагаются в следующей папке:

```text
.minecraft/config/floral/scripts/
```

<Note>
  **Hot Reloading**: Каждое сохранение файла `.lua` мгновенно перезагружает скрипт прямо в запущенном клиенте.
</Note>

## Создание первого модуля

<Steps>
  <Step title="Перейдите в папку скриптов">
    Откройте директорию `.minecraft/config/floral/scripts/` на вашем компьютере.
  </Step>
  <Step title="Создайте Lua-файл">
    Создайте текстовый файл `my_hud_module.lua`.
  </Step>
  <Step title="Добавьте исходный код">
    Вставьте пример кода для регистрации HUD-модуля:

    <CodeGroup>
      ```lua my_hud_module.lua
      -- Регистрация модуля в категории HUD
      local module = client:register_module("Custom HUD", "Отображает FPS и ник игрока", "HUD")

      -- Главный цикл рендеринга
      function on_render()
          if not minecraft:is_in_world() then return end

          local color_bg = renderer:rgba(15, 15, 20, 180)
          local color_primary = renderer:get_client_primary_color()
          local text_info = "FPS: " .. tostring(minecraft:get_fps()) .. " | User: " .. minecraft:get_username()

          -- Отрисовка блюра, фона и текста
          renderer:prepare_blur(6.0)
          renderer:draw_blur(10, 10, 220, 34, 8)
          renderer:draw_rect_rounding(10, 10, 220, 34, 8, color_bg)
          renderer:draw_text_centered_in_rect(text_info, 10, 10, 220, 34, color_primary, 1.0)
      end
      ```
    </CodeGroup>
  </Step>
  <Step title="Проверьте результат в игре">
    Сохраните файл `Ctrl + S`. Floral мгновенно активирует ваш новый модуль в игре!
  </Step>
</Steps>

<Tip>
  Перейдите к [Спецификации Lua API](/lua-api), чтобы узнать обо всех доступных функциях отрисовки текста, текстур, блюра и получения игровых данных.
</Tip>
